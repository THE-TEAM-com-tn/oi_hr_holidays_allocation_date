Show Duration in Leave Allocation

"Effortlessly Manage to Leave Allocations with Duration Display in Odoo"

This feature is a powerful tool that allows users to easily view the duration of their leave allocations during the allocation process.

Description

The "Show Duration in Leave Allocation" module is a powerful tool that provides users with an easy way to view the duration of their leave allocations. With this module, users can quickly see how much time they have allocated for different types of leave, such as vacation, sick leave, or personal leave.

This feature is especially helpful for managers who need to keep track of their employees' leave requests and ensure that the leave requests are within the company's policies. This allows users to quickly identify the total number of days or hours allocated for each type of leave, ensuring they can effectively plan their work schedules.

View

  • It provides an easy way to view the duration of each leave allocation.

Key Features

  • Displays the duration of each leave allocation in a clear and concise format.

  • Automatically updates the duration of leave allocations after any modification.

  • Enables users to modify their leave allocations easily by adding or subtracting days or hours.

  • Allows users to quickly identify the total number of days or hours allocated for each type of leave.
